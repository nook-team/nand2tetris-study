# Nand2Tetris 스터디 소개

nand2tetris 는 범용 컴퓨터 시스템부터 최신 소프트웨어 구조까지 전반적인 프로그램 구성을 처음부터 다시 제작 해보는 강의 프로젝트 입니다.

해당 강의는 주차별로 진행되며 1~6주차동안에는 하드웨어 부분을 집중합니다. 7~12주차까지는 소프트웨어 부분을 집중하여 진행됩니다.

nand 2 tetris 강의는 놀랍게도 이전 지식을 필요로 하지 않습니다. 즉 전공자나 비전공자 누구나 들을 수 있는 강의이며 컴퓨터의 구성을 쉽게 설명하는 강의임을 알 수 있습니다.
